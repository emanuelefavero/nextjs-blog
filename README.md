## Run App Locally

- clone repo
- `cd` into repo
- run `npm install`
- run `npm run dev`

## Create a new NextJS App

- `npx create-next-app APP-NAME`

#### Create a new NextJS App with TypeScript and ESLint

- `npx create-next-app APP-NAME --typescript --eslint`
